# Grama-Yatri 🚍

A community-powered rural bus tracking Android app built using Kotlin and Firebase.

## 📌 About
Grama-Yatri helps village passengers track buses in real-time through community updates. Users can report when a bus reaches a stop, and the app calculates the estimated arrival time for upcoming villages.

## ✨ Features
- Real-time bus updates
- Live ETA tracking
- Route & village stop view
- User-reported alerts
- Lightweight and fast UI

## 🛠 Tech Stack
- Kotlin
- Android Studio
- Firebase Realtime Database

## 🎯 Goal
To make rural transportation more reliable and accessible for students and workers.
